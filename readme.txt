The easy and flexible way to add an MP3 player in your Blog's sidebar.

== Description ==

This super-simple plugin allows you to easily play any MP3 as a sidebar widget on your blog. 

= Features of Simple Music =

* **Bulletproof Functionality -** Just enter the song URL and go!
* **Easy Settings -** Upload a file or find one online, choose to autoplay the track, repeat, and more!
* **Fast Customization -** Changing the options, such as picking a new song or adjusting the settings, is a snap! 

= Great For... =

* Displaying a Welcome Message to visitors
* Showing off your recent songs
* Keeping your visitors entertained!

== Installation ==

1. Upload the folder `simple-music` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets and drag 'Simple Music' to your sidebar
4. Update the settings in the widget: MP3 URL, autoplay, etc.

== Frequently Asked Questions ==

= Can I play files other than MP3s? =
No, in this beta version only MP3s are supported. New updates will be released in the near future with more functionality.

= Can I Upload My Own Songs? =
Yes, Wordpress has a built-in upload feature. Simply login to your admin panel, go to Media > Add New on the sidebar and upload your MP3. 

= Can I Add More Than One Song? =
Unfortunately no, this version only supports one track. We are working on a new release that supports a full playlist.
